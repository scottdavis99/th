Watch Scott Davis in Action

Scott Davis is a sought-after speaker on the international software conference circuit. From Dubai to Des Moines, Bangalore to Broomfield, Scott has probably been in your hometown talking about state-of-the-art software development technologies and techniques. Here are just a few examples of Scott's informative and entertaining presentation style:


The Web is Wider than You Think

Oct 9, 2012 - Mobile Developer Summit (Bangalore, India)

[Watch]
In this keynote, Scott Davis demonstrates how the web has evolved beyond a single device with a screen, a keyboard, and a mouse. "One Web; Many Screens" is the new battle cry, and web developers should be sure that their website looks just as good on a 5-inch smartphone as it does on a 10-inch tablet, a 15-inch laptop, a 27-inch desktop monitor, and a 60-inch smart tv.

Mobile HTML5 Video

Oct 9, 2012 - Mobile Developer Summit (Bangalore, India)

[Watch]
No HTML5 feature has generated as much excitement and controversy as the new native video element. In this talk, Scott Davis cuts through the hype and the skepticism and gets to the real details. Which browsers support the video element? (Here's a hint: all of 'em...)

Backbone.js on Mobile

Oct 10, 2012 - Mobile Developer Summit (Bangalore, India)

[Watch]
There is an emerging wave of client-side, component-oriented, single-page web frameworks like Backbone.js, Batman.js, and others. While Backbone isn't specifically targeted at mobile developers, it is a nearly perfect fit none-the-less. Scott Davis demonstrates how to write a component-oriented single-page app for smartphones and tablets using Backbone.js.

jQuery Mobile

Oct 10, 2012 - Mobile Developer Summit (Bangalore, India)

[Watch]
In this session, Scott Davis shows you how jQuery Mobile makes it easy to define pages, forms, lists, buttons, and all sorts of finger-friendly widgets. But more importantly, you'll see how jQuery enables you to write a single HTML5 app that runs equally well on iPhones, Android phones, Kindle Fires, iPads, and all manner of other tablets and smartphones.

CoffeeScript: JavaScript Evolved

Apr 11, 2012 - SpringOne 2GX

[Watch]
Scott Davis introduces CoffeeScript through a demo, showing how to create a Hello World application and why it is an improvement over JavaScript.

Mobile HTML5

Feb 02, 2012 - Strange Loop

[Watch]
Scott Davis explains how to prepare a website for mobile devices from small tweaks – smaller screen sizes, portrait/landscape - to using HTML5’s local storage, application cache, and remote data.

Tomorrow’s Tech Today: HTML5

Nov 17, 2010 - Strange Loop

[Watch]
Scott Davis reviews some of the most important HTML5 features: semantic elements (header, footer, nav, section, and article), form enhancements, video and mobile support, already in use today.

Hidden Web Services: Microformats and the Semantic Web

Nov 16, 2010 - Strange Loop

[Watch]
Scott Davis makes a case for semantic data, pointing out that it is currently used by major websites to improve their traffic, presenting 2 ways to add metadata to a document: RDFa and microformats.

Writing DSLs in Groovy

Jun 29, 2009 - QCon London

[Watch]
After a short introduction to DSLs, Scott Davis plays with the keyboard showing how to approach the creation of a DSL by typing working snippets of Groovy code that get executed.

Real World Web Services

Aug 29, 2007 - No Fluff, Just Stuff

[Watch]
In this presentation, Scott Davis provides a pragmatic, down-to-earth introduction to Web services as used in the real world by public sites, including SOAP-based, REST and POX-style examples.
